Dilakukan dengan melakukan konfigurasi identitas pengguna Git. Selanjutnya, dibuat folder proyek dengan nama ppw-TA2 dan dilakukan inisialisasi repository lokal.
Repository lokal bernama ppw-TA2 berhasil diinisialisasi, ditandai dengan terbentuknya folder .git. 
Konfigurasi Git atas nama Alyaa Zahra juga telah tersimpan dengan benar sehingga Git siap digunakan untuk melakukan pelacakan perubahan file.


<img width="767" height="664" alt="image" src="https://github.com/user-attachments/assets/c3ad941e-6e93-4a27-852e-9fe8249efc7f" />
Menambahkan file index.html sebagai struktur dasar website portofolio. File kemudian ditambahkan dan di-commit ke repository
Perintah git log --oneline menampilkan commit pertama pada branch master.
Setelah HTML dibuat, file style.css ditambahkan untuk memperindah tampilan website. Perintah git log --oneline menampilkan dua commit menunjukkan bahwa commit bertahap telah berhasil dilakukan


<img width="981" height="735" alt="image" src="https://github.com/user-attachments/assets/452c54c4-829a-43f4-9812-e34000fbaba7" />
Repository lokal dihubungkan dengan repository GitHub 
Kemudian dilakukan push pertama jadi repository ppw-TA2 kini terhubung dengan GitHub dan seluruh file (index.html, style.css) berhasil diunggah ke repository online.


<img width="931" height="934" alt="image" src="https://github.com/user-attachments/assets/f6a5c107-41d9-46f5-9b44-7ca65f5d110b" />
Branch baru dibuat untuk menambahkan fitur navigasi, di branch ini index.html diperbarui untuk menambahkan elemen navigasi lalu di-commit. 
Branch feature-navigation berisi update baru tanpa mengganggu branch utama master. Commit baru tercatat dengan pesan “update file index.html”.
Setelah fitur navigasi selesai, branch feature-navigation digabungkan ke master, perubahan dari feature-navigation telah tergabung ke branch utama.
kemudian branch feature-navigation dihapus untuk menjaga kerapian repository.


<img width="1001" height="827" alt="image" src="https://github.com/user-attachments/assets/46480400-4b96-4e42-845a-ab9a8b979406" />
Hasil perintah git log --oneline yaitu 
- Pembuatan struktur HTML dasar
- Penambahan file CSS
- Pembaruan file HTML dengan fitur navigasi

Pada tahap akhir ini, dilakukan proses sinkronisasi antara repository lokal dengan repository GitHub agar seluruh perubahan yang telah dilakukan dapat tersimpan secara daring.
Semua perubahan dari commit terakhir di repository lokal telah berhasil diunggah ke repository GitHub.
Dengan demikian, branch master kini sepenuhnya sinkron antara repository lokal dan remote, menandakan bahwa seluruh proses pengelolaan versi proyek telah selesai dengan baik.
